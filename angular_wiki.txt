My Angular Wiki

1. setup developement environment
install software:
- nodejs (nodejs.org)
- git (git-scm.com)
- Webstorm

install angular:
CLI: npm install -g @angular/cli


-- create new project:
CLI: ng new assignment-zippy-component
Webstorm: https://www.jetbrains.com/help/webstorm/angular.html



-- add to gitlab
1. create project in gitlab
2. setup config (only once, after git installation):
git config --global user.name "Krisztian Szep"
git config --global user.email "kipu80@gmail.com"
- check it: 
git config --list

3. Init (or reinit) git. This creates the repository. .git hidden directory created. ng new creates it automatically, so no need to run after ng new.
git init

4. connecting the local repository to the gitlab repository
git remote add origin https://gitlab.com/kipu-angular/assignment-zippy-component.git

5. add all the files to the local git repository
git add .

6. commit the added files
git commit -m "Initial commit"

7. upload to the gitlab repository
git push -u origin master

8. test
git status




-- add bootstrap
CLI: inside the project folder:
	- latest: npm install bootstrap --save
	- ^3.4.1: npm install bootstrap@3 --save
styles.css: @import "~bootstrap/dist/css/bootstrap.css";

-- 20px behúzás a szélétől:
styles.css: body { padding: 20px; }


-- run/test angular application
CLI: ng serve

create new application (project)

2. in GitLab, create new project
3. in WebStorm, create new Angular CLI project with the same name
4. in WebStorm terminal, connect the local git repository to the remote repository in GitLab
	git remote add origin https://gitlab.com/kipu-angular/assignment-zippy-component.git
(not needed)	5. in WebStorm terminal, upload to the GitLab repository
(not needed)		git push -u origin all
(not needed)	6. in WebStorm terminal, test the git status
(not needed)		git status
7. start development and have fun :)

	git add .
9. commit new modifications to git
	git commit -m "Initial commit"
10. upload to the gitlab repository
	git push -u origin master
